# Java Data Structures & Algorithms Practice

This repository contains my solutions to foundational DSA problems, focusing on writing clean, optimized code for technical interviews.

## 🧠 Key Areas Covered
- **Arrays & Hashing:** Two Sum, Contains Duplicate, Group Anagrams.
- **Strings:** Valid Palindrome, Anagram detection, String manipulation.
- **Complexity Analysis:** Every solution includes a comment on Big O time/space complexity.

## 🛠️ Goals
- Move from $O(n^2)$ to $O(n)$ or $O(\log n)$ solutions.
- Practice edge-case handling (null inputs, empty arrays).
- Improve readability using meaningful variable names.

## 🚀 Future Additions
- Linked Lists (Singly/Doubly)
- Stack and Queue implementations
- Binary Search Tree traversals
